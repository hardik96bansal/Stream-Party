User can:
    - Signup and login
    - Create rooms
    - Search rooms
    - Join rooms
    - Kick user from room
    - Send Friend Requests
    - Accept Friend Requests
    - Invite friends to rooms
    - Send Messages
    - React to message
    - Play/ pause 

Rooms 
    - Can be created
    - Can be public/ private
    - Have maximum capacity
    - Notification for new join requests
