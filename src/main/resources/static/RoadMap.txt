User can:
    - Signup and login
    - Create groups
    - Search groups
    - Join groups
    - Kick user from group
    - Send Friend Requests
    - Accept Friend Requests
    - Invite friends to groups
    - Send Messages
    - React to message
    - Play/ pause 

Groups 
    - Can be created
    - Can be public/ private
    - Have maximum capacity
    - Notification for new join requests
